## Quick Wins (Implement Now)

### 1. Add Alt Text to All Images
```tsx
<Image 
  src="/images/arabica.jpg" 
  alt="Premium Mt. Elgon Arabica Coffee Beans from Uganda - Imbari Coffee"
/>
```

### 2. Add FAQ Schema
```json
{
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "What makes Imbari Coffee different?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Imbari Coffee is Uganda's premier specialty coffee exporter..."
      }
    }
  ]
}
```

### 3. Add Product Schema
```json
{
  "@type": "Product",
  "name": "Mt. Elgon Arabica Coffee Beans",
  "description": "Premium single-origin Arabica from Uganda...",
  "offers": {
    "@type": "Offer",
    "price": "17.99",
    "priceCurrency": "USD"
  }
}
```

### 4. Create robots.txt Enhancements
```
User-agent: Googlebot
Allow: /
Crawl-delay: 0

User-agent: Bingbot
Allow: /
Crawl-delay: 0
```
