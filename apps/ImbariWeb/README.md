# ImbariWeb (Frontend)

This is the Next.js frontend for Imbari Coffee, designed for seamless integration with the ImbariWeb-Backend Stripe API.

## Features
- Product catalog and cart
- Stripe checkout via backend API
- Modern UI with Tailwind CSS

## Setup
1. Copy `.env.example` to `.env.local` and set the backend API URL.
2. Run `npm install` to install dependencies.
3. Start the dev server with `npm run dev`.
